🤖 AI-Based Body Language & Emotion Detection System

📌 Overview

This project presents an AI-powered real-time emotion and body language detection system using Computer Vision and Machine Learning techniques.

The system captures live video input through a webcam, analyzes facial expressions + body posture, and predicts human emotions such as:
	•	😊 Happy
	•	😐 Neutral
	•	😢 Sad
	•	😠 Angry
	•	😫 Stressed

Additionally, the system includes real-time analytics dashboard and Telegram alert integration for monitoring critical emotional states.

⸻

🚀 Key Features

✅ Real-time emotion detection using webcam
✅ Body posture & facial expression analysis
✅ AI-based prediction using ML models
✅ Live dashboard (charts + metrics)
✅ Emotion-based scoring system
✅ Motivational quote generator
✅ 📩 Telegram alert system (for negative emotions)
✅ CSV report download

🧠 System Architecture: 
                       Webcam Input → Frame Processing → Face & Pose Detection →
                        Feature Extraction → Emotion Prediction →
                        Dashboard + Alerts + Data Logging
🛠️ Tech Stack

Programming Language
	•	Python 🐍

Libraries & Frameworks
	•	OpenCV (Video Processing)
	•	MediaPipe (Pose Detection)
	•	NumPy, Pandas (Data Handling)
	•	Matplotlib (Visualization)
	•	Streamlit (Dashboard UI)
	•	Requests (Telegram API)
  📷 How It Works
	1.	Webcam captures live video
	2.	Frames are processed using OpenCV
	3.	Emotion is detected (ML model / simulated logic)
	4.	Score + Quote generated
	5.	Data stored in session
	6.	Dashboard displays:
	•	Emotion distribution (Pie Chart)
	•	Emotion count (Bar Graph)
	•	Metrics (Avg Score, Total Records)
